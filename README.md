# 推荐系统算法实践

岗位目标：快手推荐大模型算法工程师（OneRec）

三个项目覆盖OneRec核心要求：生成式推荐（Recall@10 38.41%）+ 强化学习偏好对齐（RLHF/DPO）+ 高效训练/推理（8倍模型压缩）+ 推荐系统排序积累（AUC 0.8983），精准匹配JD全部关键词。
