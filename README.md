# 推荐系统算法实践

> 一句话：三个项目覆盖推荐系统排序（AUC 0.8983）→ 召回（Recall@10 38.41%）→ 大模型工程（8倍模型压缩）全链路，精准匹配快手OneRec岗位核心要求。

---

## 📁 项目概览

| | 项目 | 核心任务 | 技术方案 | 关键指标 |
|---|------|---------|---------|---------|
| 1 | [DIN序列CTR预估](project1_din/) | 排序阶段 | Target Attention + 负采样 | AUC 0.8983 |
| 2 | [生成式序列推荐](project2_transformer/) | 召回阶段 | Transformer Decoder + 因果掩码 | Recall@10 38.41% |
